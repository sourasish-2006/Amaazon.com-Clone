# Amaazon.com-Clone
# 🛒 Amazon Homepage Clone

A fully responsive-ready, static front-end clone of the Amazon homepage built using pure HTML and CSS. This project focuses on UI/UX principles, CSS Flexbox layout, and modern hover interactions.

## 🚀 Live Demo
*(https://sourasish-2006.github.io/Amazon.com-Clone/)*

## 🛠️ Tech Stack
* **HTML5:** Semantic markup and structure.
* **CSS3:** Custom styling, Flexbox for layout management, and smooth transitions.
* **Icons:** FontAwesome for UI icons (search, location, cart, etc.).

## ✨ Features
* **Custom Navigation Bar:** Includes a location selector, dynamic search bar with category dropdown, and hover-triggered borders.
* **Hero Section:** Full-width background image with an overlaid message box.
* **Product Grid:** 4-column product display utilizing Flexbox and CSS box-shadows for a modern card look.
* **Comprehensive Footer:** Multi-layered footer featuring Amazon's standard quick links, back-to-top button, and copyright information.
* **Interactive UI:** Smooth transitions, hover effects, and pointer cursors on all interactive elements.
